# Quant Research Library

This project is a personal quantitative finance research library designed to support continuous learning and experimentation in quantitative modeling, numerical methods, and high-performance computing.

The core of the project is implemented in **modern C++**, providing reusable and efficient components for mathematical modeling, stochastic processes, and pricing methods.  
A **Python interface** is built on top of this core to facilitate data handling, experimentation, visualization, and integration with the scientific Python ecosystem.

## Objectives

The main goals of this repository are:

- Maintain a **practical and evolving knowledge base** in quantitative finance
- Implement **robust and reusable numerical components**
- Combine **research flexibility (Python)** with **performance (C++)**
- Build a **well-tested codebase** suitable for experimentation and prototyping
- Gradually explore **high-performance techniques**, including GPU acceleration

## Project Philosophy

The architecture follows a simple principle:

- **C++**: core algorithms, models, numerical methods, and performance-critical code
- **Python**: data manipulation, experimentation, visualization, workflows, and bindings

This separation allows both **C++ users** and **Python users** to interact with the library efficiently.

## Core Topics Covered

The library will progressively include modules related to:

- Mathematical utilities and numerical methods
- Probability and statistics
- Random number generation
- Stochastic processes
- Financial models
- Derivative products
- Pricing algorithms (Monte Carlo, PDE, etc.)
- Model calibration
- Risk metrics

## Testing Philosophy

Reliability is a core objective of the project:

- **Unit tests for every C++ module**
- **Python tests for bindings and workflows**
- **Integration tests** across the stack

## Long-Term Direction

Future developments may include:

- GPU acceleration for computational kernels
- performance benchmarking tools
- extended model libraries
- larger scale calibration and simulation frameworks

This repository is primarily a **learning and research environment**, with an emphasis on clarity, correctness, and reproducibility.
